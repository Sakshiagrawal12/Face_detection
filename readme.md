# Face Mask detection System
A comprehensive face mask detection system using Deep Learning. This project implements MobileNetV2 for real-time face mask detection via webcam.

## ✨ Features

- **Data Cleaning**: Automated pipeline to clean and prepare dataset
- **Model Architectures**:
  - MobileNetV2 (more accurate, transfer learning)
- **Real-time Detection**: Webcam integration with live statistics
- **Data Augmentation**: Improves model generalization
- **Comprehensive Evaluation**: Confusion matrices, classification reports
- **Statistics Tracking**: Real-time mask compliance statistics
